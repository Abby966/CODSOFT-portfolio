Temperature converter software

Github link

This is a developed a temperature converter tool. This handy tool allows you to convert temperatures between Celsius and Fahrenheit with ease & vice-versa. You'll see how simple it is to input a value in one temperature scale and instantly get the converted values in the other scales. Whether you're working with Celsius or Fahrenheit this converter has got you covered. This project allow me to showcase my proficiency in programming and user interface design. It was an exciting opportunity to apply my knowledge in a practical context.
